#Checklists

---

Podemos criar checklists com o auxilio de `- [ ]`não esquecer o espaço entre - e [; nem do espaço entre os dois colchetes retos.

*exemplo*

**Lista de tarefas**
- [ ] tarefa 1
- [ ] tarefa 2
- [ ] tarefa 3

Para identificar tarefa comprida basta substituir o espaço por X

*exemplo*

**Lista de tarefas**
- [ ] tarefa 1
- [X] tarefa 2
- [ ] tarefa 3
